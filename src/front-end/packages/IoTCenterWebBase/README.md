#### 安装教程

##### openatom-foundation\src\front-end  为前端文件

##### 环境需求：

```
 node 23.1.0 + 

 npm install pnpm -g (在此环境下安装pnpm包管理)
 


```

##### 指令执行:

```
在根目录（front-end）执行 pnpm install , 安装完毕后，使用pnpm dev启动服务

前端连接服务需要在 openatom-foundation\src\front-end\configuration\moduleConfiguration.json 文件中进行更改

```
